# GuardianBlockChainPHP
This is PHP SDK for Guardian Blockchain
### To Install
```./composer.phar install```
### To run unit tests
```./composer.phar test```
### To run Lint/static analysis
```./composer.phar lint```
### SDK's API can be found in
```src/GuardianFacade.php```
